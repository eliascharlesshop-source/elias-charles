# Elias Charles Image Assets

This directory contains organized image assets for the Elias Charles website. All images are on-brand and follow a consistent black and white aesthetic with palm trees, ocean waves, and urban nighttime scenes.

## Directory Structure

### `/brand/`
- `highway-lights-1.jpg` - Highway lights at night, used for main brand imagery
- `highway-lights-2.jpg` - Alternative highway scene, used for feature sections

### `/hero/`
- `streetlights-night-1.jpg` - Street lights at night, used for hero sections
- `streetlights-night-2.jpg` - Alternative street scene 
- `streetlights-night-3.jpg` - Third variation for diversity

### `/lifestyle/`
- `palm-trees-sky-1.jpg` - Palm trees against sky, lifestyle imagery
- `palm-trees-sky-2.jpg` - Alternative palm tree sky scene
- `palm-trees-sky-3.jpg` - Third variation
- `palm-trees-street-1.jpg` - Palm trees along street, urban lifestyle
- `palm-trees-street-2.jpg` - Alternative street palm scene
- `palm-trees-street-3.jpg` - Third street variation

### Root Level Ocean Images
- `ocean-bw-1.jpg` - Main hero ocean scene, black and white
- `ocean-bw-2.jpg` - Secondary ocean scene
- `ocean-bw-3.jpg` - Third ocean variation
- `ocean-bw-4.jpg` - Fourth ocean scene
- `ocean-bw-5.jpg` - Fifth ocean variation
- `ocean-bw-6.jpg` - Sixth ocean scene

## Usage Guidelines

1. **Hero Sections**: Use ocean-bw-1.jpg or streetlights-night-*.jpg
2. **Feature Sections**: Use brand/highway-lights-*.jpg or lifestyle/palm-trees-*.jpg
3. **Collection Headers**: Use ocean-bw-*.jpg for ocean/surf related, lifestyle images for fashion
4. **Fallback Images**: Use lifestyle/palm-trees-street-3.jpg instead of placeholder.svg

## Image Specs
- All images are high-resolution JPEGs
- Consistent black and white color palette
- Optimized for web use while maintaining quality
- Aspect ratios suitable for hero sections and cards
